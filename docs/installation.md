## ğŸš€ Installation Steps

### Step 1: Download the Template

1. **Via GitHub (Recommended)**
   ```bash
   git clone https://github.com/vietgoldlight-oss/candgig-blogger-template.git
   ```

2. **Via ZIP Download**
   - Go to the [GitHub repository](https://github.com/vietgoldlight-oss/candgig-blogger-template)
   - Click the green "Code" button
   - Select "Download ZIP"
   - Extract the ZIP file to your computer

### Step 2: Prepare the Template File

1. Navigate to the downloaded folder
2. Go to the `template` directory
3. Locate `optimized-theme.xml` - this is your template file
4. **Do not edit this file** unless you know what you're doing

### Step 3: Upload to Blogger

1. **Access Blogger Dashboard**
   - Go to [blogger.com](https://blogger.com)
   - Sign in to your account
   - Select your blog

2. **Navigate to Theme Section**
   - Click on **Theme** in the left sidebar
   - You'll see your current theme displayed

3. **Upload New Template**
   - Click the **Backup/Restore** button (usually represented by a gear icon or arrow)
   - In the popup, you'll see options to backup/restore
   - Click **Choose File** under "Upload"
   - Select the `optimized-theme.xml` file from the template folder
   - Click **Upload**

4. **Confirm Installation**
   - Blogger will process the template
   - You'll see a preview of the new theme
   - Click **Save** to apply the changes

1. Get your Google Analytics tracking ID

3. Find `GA_TRACKING_ID` in the code

## ğŸ› Troubleshooting Common Issues

### Issue 1: Template Not Uploading
**Symptoms:** Error message when uploading template
**Solutions:**
- Check file size (should be under 1MB)
- Ensure file is valid XML
- Try uploading during off-peak hours
- Clear browser cache and try again

### Issue 2: Images Not Loading
**Symptoms:** Broken image icons or missing images
**Solutions:**
- Check image URLs in the template
- Ensure images are hosted properly
- Verify image permissions
- Check for HTTPS/HTTP mixed content issues

### Issue 3: Mobile Menu Not Working
**Symptoms:** Navigation menu doesn't work on mobile
**Solutions:**
- Clear browser cache
- Check if JavaScript is enabled
- Verify mobile viewport settings
- Test on different mobile browsers

### Issue 4: Slow Loading Speed
**Symptoms:** Blog loads slowly
**Solutions:**
- Optimize images (use WebP format)
- Enable browser caching
- Remove unnecessary widgets
- Use lazy loading for images

### Issue 5: SEO Issues
**Symptoms:** Poor search engine visibility
**Solutions:**
- Check meta descriptions
- Verify schema markup
- Submit sitemap to search engines
- Ensure proper heading structure
